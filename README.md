## Challenges & Solutions
**Challenge 1:** Making the About section layout responsive  
**Solution:** Used CSS Grid with different column configurations for mobile, tablet, and desktop.

**Challenge 2:** Styling project cards consistently  
**Solution:** Created a reusable card class with padding, shadow, and hover effect.

**Challenge 3:** Centering hero section content  
**Solution:** Applied Flexbox to the `.hero` container with `align-items: center;` and `text-align: center;`.
